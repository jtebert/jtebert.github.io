These are some of the independent programming projects on which I am currently working or that I have completed. Many of my projects can be found on my GitHub page.

Textual Analysis

Python

This is a command line program to analyze statistics on word usage in text files. Currently, it analyzes the total number of words, most used words, and distribution of individual words throughout the text.

Future goals including analyzing words by part of speech (e.g., finding most-used adjectives), allowing comparisons between text files, and importing data from other file types.

Tetris

Racket

This was a Racket (Scheme) project for the Computer Science Fundamentals course used in previous semesters. I decided to do it anyway. It is a fully functional Tetris clone, including scoring and levels. (The code is not posted on my GitHub in case the project is used again for the class.)

To run the game, make sure that "Allow executing file as program" is checked in the file's permissions.

DNA (in progress)

Python

This program models different DNA processes such as annealing, recombination, and digestion by restriction enzymes, as well as providing a simple framework for visualizing these DNA fragments.

Future goals include importing and simulating functionality with real DNA sequences, visualizing recominant and digested DNA, and adding functionality to handle circular DNA molecules.
